# 📱 تطبيق إدارة البطاقات والروابط الشخصية

تطبيق ويب متكامل لإدارة بطاقات التواصل الاجتماعي والحسابات البنكية، مع نظام تسجيل دخول وحفظ بيانات دائم في قاعدة بيانات SQLite.

![لقطة الشاشة](screenshot.png)

---

## 🧩 المميزات

- تسجيل دخول وإنشاء حساب (اسم رباعي، هاتف، بريد، كلمة سر، تاريخ ميلاد، ولاية)
- استعادة كلمة السر عبر البريد ورقم الهاتف
- إضافة روابط لمواقع التواصل الاجتماعي (فيسبوك، انستغرام، تويتر، لينكد إن، تيك توك، سناب شات، واتساب، تيليجرام، يوتيوب)
- 
- إدارة حسابات بنكية (اسم البنك، الفرع، رقم الحساب، رقم الإشعار)
- 
- باركود QR ديناميكي يحتوي على جميع بيانات المستخدم
---

1. **استنساخ المستودع**
   ```bash
